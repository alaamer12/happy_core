Key Features
------------

1. Enum Registration
   - Register enums with the registry
   - Access registered enums by name
   - Automatic enum type validation

2. Enum Lookup
   - Look up enums by name
   - Get all registered enums
   - Check if an enum is registered

3. Basic Operations
   - Register new enums
   - Retrieve registered enums
   - Remove enums from registry
